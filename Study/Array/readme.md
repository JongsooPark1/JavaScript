## Array
---

> method



a = [10, 20, 30, 40, 50];

**push()**

맨 뒤에 원소를 추가한다. 배열 a는 원소 추가된 상태
```JavaScript
a.push(60);
```
결과

a = [10, 20, 30, 40, 50, 60];

**pop()**

맨 뒤의 원소 출력한다. 배열 a는 마지막 원소 제거된 상태
```JavaScript
a.pop();

```
결과

50

**shift()**

맨 앞의 원소를 출력한다. 배열 a는 첫번째 원소 제거된 상태
```JavaScript
a.shift();
```
결과

10

**unshift()**

맨 앞에 원소를 추가한다. 배열 a는 원소 추가된 상태
```JavaScript
a.unshift(1);
```
결과

a = [1, 10 ,20, 30, 40, 50];
